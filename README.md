## What I Learned
Through this assignment, I practiced the fundamental operations of Git and GitHub, including:
- Initializing repositories and creating commits.
- Working with branches and switching between them.
- Merging branches into the main branch.
- Creating intentional merge conflicts and resolving them.

- Adding files such as text documents and screenshots to the repo.

I gained a much clearer understanding of how branching works and why merge conflicts happen.
